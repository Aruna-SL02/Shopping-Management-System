## üöÄ Getting Started

Follow these steps to set up the project locally.

### Prerequisites
* Visual Studio (2019 or later recommended)
* XAMPP, WAMP, or MAMP installed
* MySQL Connector installed

### Installation & Run

1.  **Clone or Download** the repository to your local machine.
2.  **Database Setup:**
    * Open your local server (XAMPP/WAMP).
    * Go to `phpMyAdmin`.
    * Create a new database.
    * Import the `onlineshop.sql` file provided in the root folder.
3.  **Configure the Project:**
    * Open the `OnlineShop.sln` solution file in **Visual Studio**.
    * Check the database connection string in the code to ensure it matches your local MySQL credentials (username/password).
4.  **Run:**
    * Press `F5` or click **Start** in Visual Studio to run the application.
